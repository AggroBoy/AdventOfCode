# Advent of Code

This repo contains Kotlin solutions for Advent of Code.

Solutions are organized by year under `src/`, with utility helpers in `src/util/` and some tests under `test/`. The current tree includes solutions for 2015, 2022, 2024, and 2025.

I do this every year, as much as I can, so this repo should always be mostly up to date.

## Layout

- `src/twenty15`, `src/twenty22`, `src/twenty24`, `src/twenty25`: puzzle solutions grouped by year and day.
- `src/util`: shared helpers such as coordinates, directions, caching, and collection utilities.
- `test`: a small set of Kotlin tests for selected days.

There is no build script committed at the repo root; this is intended to be run in the IDE. These are puzzle solutions, not products! :D
